## :rocket: Getting Started

## 1. Download the repository

```shell
 $ git clone https://github.com/ErickLuizA/yfmusic.git
```

## 2. Installation

```shell
# Go into the folder
$ cd yfmusic

# Install dependencies
$ npm install
```

## 3. Execution

```shell
# Run the app on expo
$ npm start
```
